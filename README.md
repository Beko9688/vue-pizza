# vue-pizza

Шаблон для старта проекта на **Vue 3** + **Vite**.

## Что нужно

- **VS Code** + расширение **Vue (Volar)** (Vetur лучше отключить)

## Devtools (по желанию)

- Chrome / Edge / Brave: Vue.js Devtools  
- Firefox: Vue.js Devtools

## Запуск проекта

```sh
npm install
npm run dev
```

## Сборка

```sh
npm run build
```
